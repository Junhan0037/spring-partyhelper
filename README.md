# spring-partyhelper [![Build Status](https://travis-ci.org/Junhan0037/spring-partyhelper.svg?branch=master)](https://travis-ci.org/Junhan0037/spring-partyhelper)

프로젝트 개발 환경은 다음과 같습니다.

* IDE : IntelliJ
* OS : Windows10
* Springboot
* SpringMVC
* SpringDataJPA
* SpringSecurity
* JPA
* QueryDSL
* JUnit5
* PostgreSQL
* Bootstrap
* Thymeleaf
* JQuery
